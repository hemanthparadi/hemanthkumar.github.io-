## 🚀 How to Run in VS Code

### Option 1 — Live Server (Recommended)
1. Open the `hemanth-portfolio` folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **"Open with Live Server"**
4. Portfolio opens at `http://127.0.0.1:5500`

### Option 2 — Open Directly
1. Double-click `index.html`
2. Opens in your default browser

> ⚠️ Some browsers may block local images when opening HTML files directly.
> Use Live Server for best results.

## ✏️ How to Edit

| What to change         | Where                        |
|------------------------|------------------------------|
| Name, bio, links       | `index.html`                 |
| Colors, fonts, layout  | `css/style.css`              |
| Typing animation lines | `js/main.js` → `lines` array |
| Profile photo          | Replace `images/profile.jpeg`|
| CV image               | Replace `images/cv.png`      |
| Certificate images     | Replace files in `images/`   |

## 🌐 To Deploy Online (Free)

### GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: `main` branch
3. Your portfolio will be live at `https://yourusername.github.io/repo-name`

### Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop the `hemanth-portfolio` folder
3. Live instantly with a public URL
